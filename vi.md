Jump to line N:

	vi +N file.txt

Commands

	- ( and ): start/end of sentence
	- { and }: start/end of paragraph
	- W: jump word with punctionation
	- I: insert from start of line
	- s and S: substitute char/line
	- Ctrl-R: redo
	- ZZ: x
	- ZQ: q

Register

	- create register r: "r
	- refer to register r: `r

Markers

	- create marker: m KEY
	- go to marker: ' KEY
